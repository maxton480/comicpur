HOW TO ADD NEW COMIC (2 MINUTE KA KAAM, NO CODING):

1. Naya folder bana comics/ mein (jaise comics/new-comic)
2. Cover image daal: new-comic/cover.jpg
3. Naya chapter bana: new-comic/ch1.html (copy ch1.html from mera-jalwa)
4. Detail page bana: new-comic/index.html (copy mera-jalwa ka index.html, title change kar)

5. data/comics.json mein yeh add kar (last array mein):
{
  "title": "New Comic Name",
  "slug": "new-comic",
  "cover": "comics/new-comic/cover.jpg",
  "views": 0,
  "latest": "Chapter 1",
  "genre": "romance",
  "updated": "2025-11-29"
}
